Questions related to (reporting) issues.

## I have found an issue with the site, where can I report this?

If you have found an issue with the Husqy website, please join our [Discord server](https://discord.gg/W8EdsMkZRm) and open a ticket.

## I have found an issue with the documentation site, where can I report this?

If you have found an issue with the Husqy documentation website, please join our [Discord server](https://discord.gg/W8EdsMkZRm) and open a ticket.

## A translation in my language is incorrect or is missing, what do I do?

There are a few things you can do if you have found a wrong or missing translation:

- Go to our [Discord server](https://discord.gg/W8EdsMkZRm) and create a ticket to let us know;
- Go to our [translations GitHub Page](https://github.com/husqybot/translations) and contribute yourself to the translations

## I think I have found an issue with Husqy, what can I do?

If you think you have found an issue with Husqy, please open a ticket in our [Discord server](https://discord.gg/W8EdsMkZRm)!

## Something stopped working, what now?

We are very sorry if something with Husqy stopped working. Please check our changelog to see if an update changed the working of the component, if there is nothing mentioned about your problem, please use one of the following solutions:

Check this FAQ page to see if your question is already answered;
Check you documentation to see if there is an explanation that answers your question;
Join our Discord server to get all the help you need;

:::tip Check the changelog regularly!
We recommend checking this changelog regularly to prevent suprises!
:::
