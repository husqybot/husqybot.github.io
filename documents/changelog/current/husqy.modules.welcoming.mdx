### [1.0.0] - 26-01-2025

#### [BREAKING]

- Migrated the old greetings module to the new Welcoming module. Old settings will not work and the module will need to be reconfigured.

#### [NEW]

- Added different response types to welcoming module.
- Added option to randomize responses for welcome messages in DM (or select a response if you don't want to randomize).
- Added option to randomize responses for welcome messages in a channel (or select a response if you don't want to randomize).
- Added option to randomize responses for leave messages in a channel (or select a response if you don't want to randomize).
- Added support for autorole to give multiple roles when a user joins.
- Added new timedrole component to give members roles based on the amount of time they are in the server.
