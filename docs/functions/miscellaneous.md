The Husqy Miscellaneous functions provides you, your server admins and server members with tools and fun commands for your server. These commands do not belong to any specific Husqy function or module.

## The info command

The `/info {subcommand}` commands provide you with information about the Husqy configuration in your server, channels, users, roles and server information. Just run the `/info {subcommand}` command and specify, if available, the required parameter(s).

:::tip

The `/info ping` command is also the place where you can check the ping of Husqy.

:::

## The help command

The `/help` commands provides you with quick links to different Husqy resources to get help or assistance if required.

## Memes

The `/meme` command provides you and your server members to a random meme.

## Additional info & commands

:::info Available commands

These functions provides the following commands:

- `/help`;
- `/info channel`;
- `/info role`;
- `/info user`;
- `/info server`;
- `/info bot`;
- `/info ping`;
- `/meme`;

:::
